# LinkForge - URL Shortener

A feature-rich URL shortener built with Node.js, Express.js, and PostgreSQL.

##  Features

### Core Features
-  URL shortening with auto-generated 6-character codes
-  **Custom aliases** - Users can choose their own short codes
-  Redirect to original URLs with click tracking
-  **Analytics endpoint** - View detailed stats for any short URL
-  Last clicked timestamp tracking
-  PostgreSQL database with proper indexing
-  Comprehensive error handling and input validation
